About Our Project

PROJECT DESCRIPTION

Our project aims to present pandemic learning loss visually on both a national and a state level. Our data compares the average math scores of 8th graders in 2019 versus 2022. We have found that the average scores have declined in almost every single state, and hope to gain some insight into how the pandemic has affected learning for different groups of people, if time permits.
