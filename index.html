MACRO-SUMMARY

PROJECT DESCRIPTION

Our project aims to display pandemic learning loss visually on both a national and a state level. Our data compares the average math scores of 8th graders in 2019 versus 2022, and we have found that the average scores have declined in almost every single state. We hope to demonstrate insight into how the pandemic has affected learning for different demographics.
